# n8n Data Intelligence Orchestrator

## Quick start (Step 1)
1. Create `.env` from `.env.example` (if you add one later).
2. Run `docker compose up -d`.
3. n8n UI: http://localhost:5678
4. Streamlit dashboard: http://localhost:8501

This repo contains the skeleton for the MLOps orchestration challenge: collector, validator, processor, reporter workflows to be added under `/workflows`.
